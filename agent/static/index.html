Commands & Tools

Slash commands work directly in the chat input.

/helpShow this help
/clearClear conversation and memory
/save [name]Save this session with an optional name
/load [name]Load a previously saved session
/set parameter <name> <val>Adjust model parameters (e.g., temperature)
/set think / nothinkToggle reasoning mode on or off
/show parametersDisplay current model settings
/vault add <path>Index a file or folder for RAG search
/vault search <query>Search your indexed vault
/quitClose the session
