Agent Help & Capabilities

🛠 Available Tools

The agent automatically executes these tools in the background when needed:

  • Spotify Control: Play music (e.g. "play some rock music").
  • App Launcher: Open local applications (e.g. "open chrome").
  • Browser: Open URLs and default web browsers.
  • File Management: Read, write, view, and list files.
  • Knowledge Vault: Search and index local folders or documents (PDF, DOCX, etc.).

📂 Auto-indexing files

Type the absolute path of a large file (e.g. PDF or DOCX) in the chat input. The system will detect and automatically index it into the active vault before processing your request.
